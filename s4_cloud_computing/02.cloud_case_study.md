
# Cloud System Design – Descriptive Case Study Activity

## Instructions for Students

For each case:

* Identify:

  * Cloud Deployment Model (Public / Private / Hybrid)
  * Service Model (IaaS / PaaS / SaaS)

* Define:

  * Key Features Required
  * Architecture Approach
  * Risks & Challenges

* Plan:

  * **Scaling Strategy** (How the system will handle growth and traffic spikes)
  * **Cost Optimization Plan** (How to reduce and manage cloud costs efficiently)
  * **Security Enhancements** (How you will protect data and ensure secure access)

* Justify:

  * Explain clearly why your chosen solution is the most suitable for the given scenario

# Case Study 1: CodingGita Learning Platform

**Scenario:**

CodingGita is launching a full-scale online learning platform for engineering students across India. The platform includes:

* Recorded video lectures (HD streaming)
* Live coding classes with 500–2000 concurrent students
* Assignment submissions and evaluation
* Real-time chat during sessions
* Hackathon events where traffic spikes suddenly

**Constraints:**

* Budget is limited initially (startup phase)
* Must handle sudden traffic spikes during hackathons
* Students from tier-2 and tier-3 cities (network variability)
* Platform should scale in the next 2–3 years

**Student Task:**

* Decide the best cloud deployment and service model
* Design how video streaming and live sessions will work
* Suggest scaling strategy during peak load

**Focus Areas:**

* Auto-scaling
* Load balancing
* CDN usage
* Storage optimization

# Case Study 2: Social Media Startup

**Scenario:**

A startup is building a social media platform similar to Instagram where users can:

* Upload photos and videos
* Like, comment, and share content
* Receive real-time notifications
* Follow other users

The app is expected to grow rapidly if it gains traction.

**Constraints:**

* Initially small team (5 developers)
* Need fast development and deployment
* System must handle unpredictable traffic growth
* Users expect fast content loading

**Student Task:**

* Choose between IaaS and PaaS
* Design storage for media (images/videos)
* Plan for scaling when users increase from 1,000 → 1 million

**Focus Areas:**

* Horizontal scaling
* Database design (NoSQL vs SQL thinking)
* CDN
* High availability
