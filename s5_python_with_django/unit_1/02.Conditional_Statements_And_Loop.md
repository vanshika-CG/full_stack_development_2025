# Conditional Statements and Loops

---

## 1. Operators Used in Conditions

Before writing conditional statements, it helps to know the operators that produce the `True` or `False` values conditions rely on.

### 1.1 Comparison Operators

| Operator | Meaning | Example | Result |
|---|---|---|---|
| `==` | Equal to | `5 == 5` | `True` |
| `!=` | Not equal to | `5 != 3` | `True` |
| `>` | Greater than | `7 > 4` | `True` |
| `<` | Less than | `7 < 4` | `False` |
| `>=` | Greater than or equal to | `5 >= 5` | `True` |
| `<=` | Less than or equal to | `4 <= 3` | `False` |

```python
x = 10
y = 20
print(x == y)   # Output: False
print(x < y)    # Output: True
```

### 1.2 Logical Operators

Logical operators combine multiple conditions into a single result.

| Operator | Meaning | Example |
|---|---|---|
| `and` | True only if both conditions are true | `(5 > 2) and (3 > 1)` → `True` |
| `or` | True if at least one condition is true | `(5 > 2) or (3 < 1)` → `True` |
| `not` | Reverses the result of a condition | `not (5 > 2)` → `False` |

```python
age = 25
has_id = True

if age >= 18 and has_id:
    print("Entry allowed")
```

**Explanation:** Both `age >= 18` and `has_id` must be `True` for the message to print, since `and` requires every condition to be true.

## 2. Conditional Statements

### Definition

A **conditional statement** is a programming construct that executes a block of code only if a specified condition evaluates to `True`.

### 2.1 The `if` Statement

```python
age = 20

if age >= 18:
    print("You are eligible to vote")
```

**Explanation:**

- `if age >= 18:` checks whether the condition is true.
- The indented line below only runs if the condition holds.
- If the condition is `False`, the block is simply skipped.

### 2.2 The `if...else` Statement

Used when one block should run if the condition is true, and a different block should run if it is false.

```python
age = 15

if age >= 18:
    print("You are eligible to vote")
else:
    print("You are not eligible to vote")
```

### 2.3 The `if...elif...else` Statement

Used when there are more than two possible outcomes to check.

```python
marks = 72

if marks >= 90:
    grade = "A"
elif marks >= 75:
    grade = "B"
elif marks >= 60:
    grade = "C"
else:
    grade = "F"

print("Grade:", grade)   # Output: Grade: C
```

**Explanation:**

- Python checks each `elif` condition in order, from top to bottom.
- As soon as one condition is `True`, its block runs and the rest are skipped entirely.
- `else` only runs if none of the preceding conditions were true.

### 2.4 Nested Conditional Statements

A conditional statement placed inside another conditional statement is called a **nested if**.

```python
num = 15

if num > 0:
    if num % 2 == 0:
        print("Positive and Even")
    else:
        print("Positive and Odd")
else:
    print("Not a positive number")
```

**Explanation:**

- The outer `if` checks whether `num` is positive.
- Only if that is true does Python evaluate the inner `if...else`, which checks whether the number is even or odd using the modulus operator `%`.

> **Important — Common Mistake**
>
> Deep nesting (if inside if inside if) makes code hard to read and debug. Where possible, combine conditions using `and` / `or` instead of nesting multiple levels.

## 3. Loops

### Definition

A **loop** is a programming construct that repeatedly executes a block of code as long as a specified condition is met, or for a fixed number of iterations.

Python provides two main types of loops: `for` and `while`.

### 3.1 The `for` Loop

A `for` loop is used to iterate over a sequence — such as a list, string, or range of numbers — executing the loop body once for each item.

```python
for i in range(5):
    print(i)
```

**Output:**

```text
0
1
2
3
4
```

**Explanation:**

- `range(5)` generates a sequence of numbers from `0` up to, but not including, `5`.
- `i` takes on each value in that sequence, one at a time, on each pass through the loop.

### Understanding `range()`

| Form | Behavior | Example |
|---|---|---|
| `range(stop)` | Starts at `0`, ends before `stop` | `range(5)` → `0,1,2,3,4` |
| `range(start, stop)` | Starts at `start`, ends before `stop` | `range(2, 6)` → `2,3,4,5` |
| `range(start, stop, step)` | Increments by `step` each time | `range(0, 10, 2)` → `0,2,4,6,8` |

```python
for i in range(2, 10, 2):
    print(i)
```

**Output:**

```text
2
4
6
8
```

### Looping Through a String

```python
for letter in "Python":
    print(letter)
```

**Explanation:** A string is a sequence of characters, so a `for` loop visits each character in order, from left to right.

### Looping Through a List

```python
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
```

---

### 3.2 The `while` Loop

A `while` loop repeats a block of code **as long as** a given condition remains `True`.

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

**Output:**

```text
1
2
3
4
5
```

**Explanation:**

- The condition `count <= 5` is checked before every iteration.
- `count += 1` increases `count` by `1` each time, which is essential — without it, the condition would always remain true.

> **Important — Common Mistake**
>
> Forgetting to update the loop variable inside a `while` loop creates an **infinite loop**, since the condition never becomes `False`. Always double-check that something inside the loop moves the condition toward becoming false.

### `for` vs `while`: When to Use Which

| Situation | Preferred Loop |
|---|---|
| Number of iterations is known in advance | `for` |
| Iterating over a sequence (list, string, etc.) | `for` |
| Looping until a condition changes, with an unknown number of iterations | `while` |
| Repeating until user provides specific input | `while` |

> **Important — Interview Tip**
>
> A common interview question: *"Can every `for` loop be written as a `while` loop?"*
> Yes — a `for` loop is essentially a `while` loop with built-in iteration handling. Interviewers sometimes ask you to manually convert one into the other to test your understanding of how iteration actually works underneath.

## 4. Loop Control Statements

Python provides three statements that alter the normal flow of a loop: `break`, `continue`, and `pass`.

### 4.1 `break`

#### Definition

`break` immediately terminates the loop it is inside, skipping any remaining iterations.

```python
for num in range(1, 10):
    if num == 5:
        break
    print(num)
```

**Output:**

```text
1
2
3
4
```

**Explanation:** As soon as `num` equals `5`, the `break` statement exits the loop entirely — `5` through `9` are never printed.

### 4.2 `continue`

#### Definition

`continue` skips the rest of the current iteration and moves directly to the next one, without exiting the loop.

```python
for num in range(1, 6):
    if num == 3:
        continue
    print(num)
```

**Output:**

```text
1
2
4
5
```

**Explanation:** When `num` equals `3`, `continue` skips the `print(num)` line for that iteration only, then the loop carries on normally with `4` and `5`.

### 4.3 `pass`

#### Definition

`pass` is a null operation — it does nothing and acts purely as a placeholder where Python syntax requires a statement.

```python
for num in range(5):
    if num == 3:
        pass   # placeholder - to be implemented later
    print(num)
```

**Explanation:** `pass` is commonly used while writing the skeleton of a program, allowing an empty `if` block, loop, or function to exist without causing a `SyntaxError`, until the actual logic is added later.

## 5. The `else` Clause with Loops

Python allows an `else` block after a `for` or `while` loop. This `else` runs only if the loop completes **without** being interrupted by a `break`.

```python
for num in range(1, 5):
    print(num)
else:
    print("Loop finished without a break")
```

**Output:**

```text
1
2
3
4
Loop finished without a break
```

If a `break` occurs, the `else` block is skipped entirely:

```python
for num in range(1, 5):
    if num == 3:
        break
    print(num)
else:
    print("This will not print")
```

**Output:**

```text
1
2
```

## 6. Nested Loops

A loop placed inside another loop is called a **nested loop**. The inner loop completes all of its iterations for each single iteration of the outer loop.

```python
for i in range(1, 4):
    for j in range(1, 3):
        print(f"i={i}, j={j}")
```

**Output:**

```text
i=1, j=1
i=1, j=2
i=2, j=1
i=2, j=2
i=3, j=1
i=3, j=2
```

**Explanation:** For each value of `i` (the outer loop), the inner loop runs completely through all values of `j` before `i` advances to its next value.

### Practical Example: Printing a Pattern

```python
rows = 5

for i in range(1, rows + 1):
    print("*" * i)
```

**Output:**

```text
*
**
***
****
*****
```

**Explanation:** On each iteration, `"*" * i` repeats the asterisk character `i` times, producing one extra `*` per row.
