## UNIT 5 — Spring Boot + MySQL (JPA)

**Q30.** Build a complete **Student CRUD REST API** with Spring Boot + MySQL using 3-layer architecture:

**Files to create:**
1. `Student.java` — Entity
2. `StudentRepository.java` — extends `JpaRepository`
3. `StudentService.java` — `@Service` with Constructor Injection
4. `StudentController.java` — `@RestController`

**Endpoints:**

| Method | URL | Action | Status |
|--------|-----|--------|--------|
| POST | `/api/students` | Add student | 201 |
| GET | `/api/students` | Get all | 200 |
| GET | `/api/students/{id}` | Get by ID | 200 / 404 |
| PUT | `/api/students/{id}` | Update | 200 / 404 |
| DELETE | `/api/students/{id}` | Delete | 200 / 404 |

Test all endpoints in **Postman**. Verify records appear in MySQL using `SELECT * FROM students;`.
