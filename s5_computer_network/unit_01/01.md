# Introduction to Computer Networks
## Network History, Introduction to Network, and Uses of Computer Networks

---

## 1. Network History

### Origin

- Before computer networks existed, data had to be physically transferred using magnetic tapes, floppy disks, or printed documents.
- This method was slow, expensive, and error-prone.
- The need for faster and reliable data sharing led to the development of computer networks.

### Timeline of Key Developments

**1960s - Birth of Networking (ARPANET)**
- The United States Department of Defense funded a project called ARPANET (Advanced Research Projects Agency Network).
- ARPANET connected four universities in the US: UCLA, Stanford, UCSB, and University of Utah.
- It is considered the first operational packet-switching network and the direct ancestor of the modern internet.
- The primary goal was to allow researchers to share data and to maintain communication even if one part of the network was destroyed.

**1970s - Email and Protocol Development**
- Ray Tomlinson sent the first network email in 1971 using ARPANET.
- The "@" symbol was introduced to separate the username from the host machine name.
- NCP (Network Control Protocol) was the first standard protocol used on ARPANET.
- Ethernet technology was developed by Xerox in 1973, which became the foundation of Local Area Networks (LAN).

**1980s - TCP/IP and Expansion**
- TCP/IP (Transmission Control Protocol / Internet Protocol) was standardized in 1983.
- TCP/IP became the universal language for computers to communicate over a network.
- The Domain Name System (DNS) was introduced in 1984 to replace numeric IP addresses with readable names (e.g., google.com).
- NSFNET (National Science Foundation Network) was launched, expanding internet access to more universities.

**1990s - World Wide Web**
- Tim Berners-Lee invented the World Wide Web (WWW) in 1989 at CERN, Switzerland.
- The first web browser (Mosaic) was released in 1993, making the internet accessible to the general public.
- The internet became commercially available, and the number of users grew rapidly.

**2000s Onwards - Modern Era**
- Wireless networks (Wi-Fi), broadband internet, and mobile networks (3G, 4G, 5G) transformed connectivity.
- Cloud computing, social media, IoT (Internet of Things), and streaming services became possible because of advanced networking.
- Today, billions of devices across the world are interconnected through the internet.

## 2. Introduction to Computer Networks

### Definition

A computer network is defined as a collection of two or more interconnected computing devices (such as computers, printers, servers, and mobile devices) that are linked together using communication channels (wired or wireless) to share data, resources, and services.

### Key Points of the Definition

- A network requires at least two devices to be formed.
- Devices are connected through a communication medium (cable or wireless).
- The main purpose is sharing of data and resources.
- Communication between devices follows a set of rules called protocols.

### Basic Components of a Computer Network

**1. Node (End Device)**
- Definition: Any device that is connected to a network and can send or receive data.
- Examples: Computer, laptop, mobile phone, tablet, printer, server.

**2. Communication Medium (Link)**
- Definition: The physical or wireless path through which data travels from one node to another.
- Types:
  - Wired Medium: Twisted pair cable, coaxial cable, optical fiber cable.
  - Wireless Medium: Radio waves, infrared, microwave, satellite.

**3. Sender**
- Definition: The device that originates and transmits the data.
- Example: A computer sending an email.

**4. Receiver**
- Definition: The device that receives and processes the transmitted data.
- Example: The recipient's email server receiving the email.

**5. Protocol**
- Definition: A set of rules and standards that govern how data is transmitted and received between devices in a network.
- Without a protocol, two devices cannot understand each other even if they are physically connected.
- Examples: TCP/IP, HTTP, FTP, SMTP.

**6. Network Interface Card (NIC)**
- Definition: A hardware component installed in a device that allows it to connect to a network.
- Each NIC has a unique MAC (Media Access Control) address.

**7. Switch / Hub / Router**
- These are intermediate devices that help direct data from one node to another within or across networks.
- Switch: Connects devices within the same network.
- Router: Connects two or more different networks.

### Characteristics of a Good Computer Network

1. **Performance** - The network should transfer data at high speed with minimum delay.
2. **Reliability** - The network should be available at all times and should recover quickly from failures.
3. **Scalability** - New devices should be able to be added to the network without affecting performance.
4. **Security** - Data should be protected from unauthorized access, modification, or loss.
5. **Fault Tolerance** - The network should continue to function even if one part of it fails.

### What is Data Communication?

- Data communication is the process of transferring data (text, audio, video, images) between two or more devices through a transmission medium.
- For successful data communication, the following five components must be present:
  1. **Message** - The actual data or information to be transmitted.
  2. **Sender** - The device that sends the message.
  3. **Receiver** - The device that receives the message.
  4. **Medium** - The path through which the message travels.
  5. **Protocol** - The rules that govern the communication.

## 3. Uses of Computer Networks

### 1. Resource Sharing

- Definition: Resource sharing means allowing multiple users on a network to use the same hardware, software, or data without duplicating it on each machine.
- Resources that can be shared include: printers, scanners, storage devices, internet connections, and software.
- Benefit: Reduces cost because one device can serve many users.
- Example: In a college computer lab, 30 computers share one printer and one internet connection through a network.

### 2. Data and File Sharing

- Definition: Networks allow users to transfer files and data from one device to another quickly and efficiently.
- Without a network, files must be transferred using physical storage devices (pen drive, CD).
- Example: A teacher uploads lecture notes on a server; students can download those notes on their own devices from any location.

### 3. Communication

- Networks enable fast and reliable communication between individuals and organizations.
- Types of communication supported by networks:
  1. **Email (Electronic Mail)** - Sending text messages and file attachments electronically. Example: Gmail, Outlook.
  2. **Instant Messaging** - Real-time text communication. Example: WhatsApp, Telegram.
  3. **Video Conferencing** - Face-to-face communication over the internet. Example: Zoom, Google Meet.
  4. **VoIP (Voice over Internet Protocol)** - Voice calls made over the internet. Example: WhatsApp calls, Skype.

### 4. Centralized Data Storage and Management

- Definition: All data is stored on a central server and all authorized users on the network can access it.
- Benefit: Easier to manage, backup, and secure data in one place rather than on individual computers.
- Example: In a hospital, all patient records are stored on a central server. Doctors and nurses access them through the hospital network from any department.

### 5. Internet Access

- Networks connect individual devices to the global internet.
- Through the internet, users can browse websites, stream videos, access cloud applications, do online banking, and use social media.
- Example: A home router connects all devices (laptop, phone, smart TV) in a house to the internet through a single broadband connection.

### 6. Remote Access

- Definition: Remote access allows a user to connect to and control a computer or server that is physically located at a different place.
- Example: A software engineer working from home can remotely connect to the company's office server and access all required files and applications through a VPN (Virtual Private Network).

### 7. E-Commerce

- Definition: E-commerce means buying and selling of goods and services over the internet, which is made possible through computer networks.
- Every action on an e-commerce platform (searching a product, placing an order, making payment) happens over a network.
- Example: Placing an order on Amazon or Flipkart involves communication between the user's device, the company's server, and the bank's payment server — all through a network.

### 8. Entertainment and Media

- Networks support online streaming, gaming, and content delivery platforms.
- Example: Watching a movie on Netflix requires continuous data transfer over the network from Netflix's servers to the user's device.

### 9. Education (E-Learning)

- Networks enable online education through LMS platforms, video lectures, and virtual classrooms.
- Example: Platforms like Coursera, NPTEL provide course content to students anywhere in the world through the internet.

## Important Definitions for Exam (Quick Revision)

- **Computer Network:** A collection of two or more interconnected devices that communicate with each other to share data and resources using a communication medium and protocols.

- **Protocol:** A set of rules that defines how data is formatted, transmitted, and received between devices on a network.

- **Node:** Any device connected to a network that can send or receive data.

- **Data Communication:** The process of transferring data between two or more devices through a transmission medium following a set of agreed rules.

- **NIC (Network Interface Card):** A hardware component that connects a device to a network and provides it with a unique MAC address.

- **ARPANET:** Advanced Research Projects Agency Network; the first packet-switching network, developed in the 1960s by the US Department of Defense; it is the predecessor of the modern internet.

- **WWW (World Wide Web):** A system of interlinked hypertext documents and multimedia content accessed via the internet; invented by Tim Berners-Lee in 1989.

- **TCP/IP:** Transmission Control Protocol / Internet Protocol; the standard set of rules that governs how data is sent and received over the internet.

- **Resource Sharing:** The ability of multiple network users to access and use common hardware, software, or data from a single shared source.

- **Remote Access:** The ability to connect to and control a computer or server from a physically different location through a network.
