# Divide and Conquer Matrix Multiplication


# 13. Mathematical Derivation of Divide and Conquer Matrix Multiplication

In **Part 1A**, we learned that a large matrix can be divided into four equal-sized sub-matrices.

Now, the next question is:

> **After dividing the matrices into four parts, how do we multiply them?**

To answer this, we first derive the mathematical formula used in Divide and Conquer Matrix Multiplication.


## Step 1: Consider Two Large Matrices

Suppose we have two matrices

```
A × B
```

where both matrices are of size

```
4 × 4
```

Matrix **A**

```
┌───────────────┐
│               │
│      A        │
│               │
└───────────────┘
```

Matrix **B**

```
┌───────────────┐
│               │
│      B        │
│               │
└───────────────┘
```

Instead of multiplying these matrices directly,

we divide each matrix into four equal sub-matrices.


## Step 2: Divide Matrix A

After partitioning,

Matrix **A** becomes

```
           Matrix A

┌─────────┬─────────┐
│   A11   │   A12   │
├─────────┼─────────┤
│   A21   │   A22   │
└─────────┴─────────┘
```

where

```
A11 = Top Left Matrix

A12 = Top Right Matrix

A21 = Bottom Left Matrix

A22 = Bottom Right Matrix
```

Each block is itself a smaller matrix.

---

## Step 3: Divide Matrix B

Similarly,

Matrix **B** becomes

```
           Matrix B

┌─────────┬─────────┐
│   B11   │   B12   │
├─────────┼─────────┤
│   B21   │   B22   │
└─────────┴─────────┘
```

Again,

each block is itself a smaller matrix.


# 14. Representing the Matrices

Instead of writing the complete matrices,

we simply represent them as

```
        ┌───────────────┐
        │ A11     A12   │
A   =   │               │
        │ A21     A22   │
        └───────────────┘
```

Similarly,

```
        ┌───────────────┐
        │ B11     B12   │
B   =   │               │
        │ B21     B22   │
        └───────────────┘
```

Suppose the result matrix is

```
C = A × B
```

Then,

```
        ┌───────────────┐
        │ C11     C12   │
C   =   │               │
        │ C21     C22   │
        └───────────────┘
```

Our goal is to calculate

```
C11

C12

C21

C22
```


# 15. How is C11 Calculated?

To calculate

```
C11
```

observe the standard matrix multiplication rule.

The **first row** of Matrix A is multiplied by the **first column** of Matrix B.

```
Row 1

↓

A11   A12
```

```
Column 1

↓

B11

B21
```

Now apply ordinary matrix multiplication.

```
(A11 × B11)

+

(A12 × B21)
```

Therefore,

```
C11

=

A11B11

+

A12B21
```



# 16. How is C12 Calculated?

Now calculate

```
C12
```

Take

```
First Row

↓

A11

A12
```

Multiply with

```
Second Column

↓

B12

B22
```

Therefore,

```
C12

=

A11B12

+

A12B22
```


# 17. How is C21 Calculated?

Now move to the second row.

Take

```
A21

A22
```

Multiply with

```
B11

B21
```

Therefore,

```
C21

=

A21B11

+

A22B21
```


# 18. How is C22 Calculated?

Finally,

```
Second Row

↓

A21

A22
```

Multiply with

```
Second Column

↓

B12

B22
```

Therefore,

```
C22

=

A21B12

+

A22B22
```


# 19. Final Divide and Conquer Formula

Combining all four equations,

we obtain

```
C11

=

A11B11

+

A12B21
```

```
C12

=

A11B12

+

A12B22
```

```
C21

=

A21B11

+

A22B21
```

```
C22

=

A21B12

+

A22B22
```

These four equations form the foundation of Divide and Conquer Matrix Multiplication.

Notice that every multiplication involves **smaller matrices** rather than the original large matrix.

## Step 3

Compute

```

A11B11

```

Notice something important.

We are again multiplying matrices.

```

```

Similarly,

```

```

Therefore,

the algorithm calls itself recursively.


# 21. Recursive Nature

Suppose

```
A11

↓

2 ×2
```

Again,

it can be divided into

```
1 ×1
```

matrices.

```
2 ×2

↓

1 ×1

1 ×1

1 ×1

1 ×1
```

Again,

the same formulas are applied.

Eventually,

every multiplication becomes

```
Number × Number
```

At that point,

recursion stops.

This smallest problem is called the **Base Case**.


# 22. Divide and Conquer Recursion Tree

```
4 ×4 Matrix

          │

Divide

          │

────────────────────────

2 ×2

2 ×2

2 ×2

2 ×2

          │

Again Divide

          │

────────────────────────

1 ×1

1 ×1

1 ×1

1 ×1

          │

Multiply

          │

Return

          │

Combine

          │

Final Matrix
```

The recursion continues until each matrix contains only one element.


# 23. Why Does This Approach Work?

Notice that

```
4 ×4
```

matrix multiplication

is transformed into

```
2 ×2
```

matrix multiplication.

Again,

each

```
2 ×2
```

matrix multiplication becomes

```
1 ×1
```

matrix multiplication.

Thus,

the original problem is repeatedly reduced into smaller problems of the same type.

This perfectly follows the Divide and Conquer paradigm.

```
Divide

↓

Solve Smaller Problems

↓

Combine
```



# 24. Important Observations

While studying the mathematical derivation, remember the following points.

- Every large matrix is divided into **four equal sub-matrices**.
- Each sub-matrix is itself a valid matrix.
- The result matrix is also divided into four corresponding blocks.
- Each block of the result matrix is computed using two smaller matrix multiplications and one matrix addition.
- The same multiplication process is repeated recursively until **1 × 1** matrices are reached.


# 25. Important Definitions

**Quadrant Matrix:**  
One of the four equal-sized sub-matrices obtained after partitioning a larger matrix.

**Recursive Matrix Multiplication:**  
A technique in which each matrix multiplication is further divided into smaller matrix multiplications until the base case is reached.

**Base Case:**  
The smallest sub-problem in recursion. For matrix multiplication, the base case occurs when the matrices become **1 × 1**, requiring only a single number multiplication.

**Result Matrix:**  
The matrix obtained after multiplying two matrices.



# 26. Summary

- Large matrices are divided into four equal-sized sub-matrices.
- The resulting matrix is also represented using four sub-matrices.
- Four fundamental equations (**C11, C12, C21, C22**) are derived using the standard row-by-column multiplication rule.
- Each multiplication in these equations involves smaller matrices.
- The algorithm recursively repeats the same process until the matrices become **1 × 1**.
- This recursive decomposition is the core idea behind Divide and Conquer Matrix Multiplication and provides the foundation for **Strassen's Matrix Multiplication Algorithm**.

#include <iostream>
#include <vector>

using namespace std;

typedef vector<vector<int>> Matrix;

// Function to Add Two Matrices
Matrix add(Matrix A, Matrix B)
{
    int n = A.size();
    Matrix C(n, vector<int>(n));

    for(int i = 0; i < n; i++)
    {
        for(int j = 0; j < n; j++)
        {
            C[i][j] = A[i][j] + B[i][j];
        }
    }

    return C;
}

// Divide and Conquer Matrix Multiplication
Matrix multiply(Matrix A, Matrix B)
{
    int n = A.size();

    // Base Case
    if(n == 1)
    {
        return {{A[0][0] * B[0][0]}};
    }

    int half = n / 2;

    Matrix A11(half, vector<int>(half));
    Matrix A12(half, vector<int>(half));
    Matrix A21(half, vector<int>(half));
    Matrix A22(half, vector<int>(half));

    Matrix B11(half, vector<int>(half));
    Matrix B12(half, vector<int>(half));
    Matrix B21(half, vector<int>(half));
    Matrix B22(half, vector<int>(half));

    // Divide Matrix A and B into 4 parts
    for(int i = 0; i < half; i++)
    {
        for(int j = 0; j < half; j++)
        {
            A11[i][j] = A[i][j];
            A12[i][j] = A[i][j + half];
            A21[i][j] = A[i + half][j];
            A22[i][j] = A[i + half][j + half];

            B11[i][j] = B[i][j];
            B12[i][j] = B[i][j + half];
            B21[i][j] = B[i + half][j];
            B22[i][j] = B[i + half][j + half];
        }
    }

    // Recursive Calls
    Matrix C11 = add(multiply(A11, B11), multiply(A12, B21));

    Matrix C12 = add(multiply(A11, B12), multiply(A12, B22));

    Matrix C21 = add(multiply(A21, B11), multiply(A22, B21));

    Matrix C22 = add(multiply(A21, B12), multiply(A22, B22));

    // Combine the four sub-matrices
    Matrix C(n, vector<int>(n));

    for(int i = 0; i < half; i++)
    {
        for(int j = 0; j < half; j++)
        {
            C[i][j] = C11[i][j];
            C[i][j + half] = C12[i][j];
            C[i + half][j] = C21[i][j];
            C[i + half][j + half] = C22[i][j];
        }
    }

    return C;
}

int main()
{
    Matrix A =
    {
        {1,2,3,4},
        {5,6,7,8},
        {9,1,2,3},
        {4,5,6,7}
    };

    Matrix B =
    {
        {2,1,3,4},
        {5,6,7,8},
        {9,8,1,2},
        {3,4,5,6}
    };

    Matrix C = multiply(A, B);

    cout << "Result Matrix\n\n";

    for(auto row : C)
    {
        for(auto x : row)
            cout << x << " ";

        cout << endl;
    }

    return 0;
}
